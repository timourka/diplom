# предполагаемая идея диплома:
Разработать мобильное приложение, которое помогает пользователю найти дату изготовления на продукте.

# предполагаемый алгоритм использования:
наводишь камеру на продукт -> подсвечивается предполагаемое место с датой
если приложение не находит дату -> нажать кнопку "сообщить об ошибке" -> снимать упаковку тыкая пальцем на место с датой

# предполагаемая структура ИС:
### мобильное приложение клиент  
обученая модель для распознования объектов запускается в приложении и ищит на изображении с камеры объект дату

### инфромация сервер -> клиент  
данные пользователя 
обученая модель  

### информация клиент -> сервер
сообщения об ошибке включающие в себя размеченные новые данные

### серверная часть приложения  
хранит данные пользователей, дообучает модель в случае появлении новых данных
